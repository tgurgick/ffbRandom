FANTASY FOOTBALL
DIVISION RANDOMIZER

USE | Randomly assign a list of team or manager names to one of two divisions

VER | 1.1

PUB | 8.17.19


HOW

  1. Enter Names into text box below and click 'RUN'
  2. Javascript combines inputs into an arary
  3. Array is then randomized via the Fisher-Yates Shuffle
  4. After randomized, a helper function splits into two groups
  5. results are displayed as output

Note: this is only designed to split into two divisions. An additional function would need to be added to take in a variable amount of divisions in the future.
